📊 TN 2026 Election Prediction using Machine Learning

This project focuses on predicting the outcome of the Tamil Nadu 2026 Legislative Assembly Election using historical election data, machine learning models, and data-driven insights.

🚀 Project Overview

The model is trained on constituency-level data from past elections (1971–2021) to forecast potential winners for the 2026 election. It incorporates political, demographic, and historical voting patterns to simulate real-world election scenarios.

Tamil Nadu 2026 is expected to be a multi-cornered contest with major alliances like DMK+, AIADMK+, and emerging players like TVK influencing vote share and seat distribution .

🧠 Key Features
📌 Constituency-level prediction (234 seats)
📊 Data preprocessing & feature engineering
🤖 ML models:
Random Forest
XGBoost
Gradient Boosting
🔁 Cross-validation for accuracy evaluation
📉 Visualization of trends and predictions
📂 Dataset
Historical election data (1971–2021)
Alliance & party-level vote share
Incumbency & anti-incumbency factors
Margin of victory
Demographic indicators
⚙️ Key Engineered Features
Rolling win rates (party-wise)
Anti-incumbency indicator
Previous margin strength (normalized)
Alliance-based vote transfer
Year/Trend encoding
📈 Model Output
Predicted winner for each constituency
Seat distribution across alliances
Confidence scores for predictions
Identification of toss-up seats (close contests)

Example insight: Around 38 constituencies are highly competitive, where even small vote swings can change results .

🎯 Objective
Understand electoral patterns using data science
Simulate different political scenarios
Provide insights into swing seats and voter behavior

⚠️ Disclaimer

This project is for educational and analytical purposes only. Predictions are based on historical data and model assumptions—they do not represent actual election results.
